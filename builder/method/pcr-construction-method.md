# PCR Construction Method

PCR records are canonical methodology assets. A classification code may seed a PCR record or map to an existing PCR record, but the classification code is not the PCR identity.

Initial lifecycle:

1. Normalize classification source data.
2. Map classification leaves to existing or new canonical PCR ids.
3. Scaffold missing PCR markdown records.
4. Validate frontmatter, modules, mappings, and catalog consistency.
5. Render PCR records for agent and CLI consumption.
