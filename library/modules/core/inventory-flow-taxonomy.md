# Inventory Flow Taxonomy

This module defines product, waste, and elementary flow treatment in PCR process inventories.

PCR inventory content is organized by process, then by direction, then by flow type:

- process
- inputs or outputs
- product, waste, or elementary flows
- individual flow rows

This hierarchy is the default structure for both Markdown PCRs and `structured.yaml`.
